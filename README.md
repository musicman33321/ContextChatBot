Description:

A fashion-trend chatbot that provides more tailored responses given a local dataset of 2023 fashion trends. Jupyter notebook is converted to usable uv package that can be run via command-line using your own OpenAI API key.


Requirements:

- uv installed on local system (command-line version only)

- OpenAI API key in .env file within src/contextchatbot directory


Usage:

- Provide OpenAI API key in .env file

- To run, use "uv run cli.py <fashion trend query>"

- Chatbot returns original query, response w/o context, response with added context, and tokens used# ContextChatBot
